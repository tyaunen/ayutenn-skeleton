# ayutenn データベーススキル

このプロジェクトは **ayutenn-core** フレームワークの宣言的マイグレーション機能を使用しています。
テーブルを作成・変更する際は、以下の手順に従ってください。

## 手順

### テーブルの作成・変更
1. `tables/` にJSON形式でテーブル定義を作成
2. `format` キーでバリデーションルールと連携可能（齟齬防止）
3. `php vendor/bin/migrate.php` でマイグレーションSQL生成
4. 生成されたSQLを確認してデータベースに適用
- 詳細: `references/table.md` を参照

## 制約
- テーブル操作は必ず宣言的マイグレーションを使用すること（手動SQL禁止）
- テーブル定義作成前にバリデーションルールの有無をチェックし、`format` キーで連携すること
- 詳細な型定義・オプションは必ず `references/table.md` を確認すること
